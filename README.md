# dmOverlay

> Shows overlay notifications for friend DMs on top of games.

## Features

- **In-Game Overlay**: Receive messsages from friends directly while in-game.
- **Quick Reply**: Send replies without alt-tabbing.
- **Friend Filter**: Only notifies you for DMs from friends (ignores strangers/bots).
- **Sound Notifications**: Plays a sound when a new DM arrives.

## Installation

This is a [Vencord](https://vencord.dev/) plugin.

1. Clone this repository into your Vencord userplugins directory:
   ```bash
   git clone https://github.com/neozepron/dmOverlay.git src/userplugins/dmOverlay
   ```
2. Rebuild Vencord:
   ```bash
   pnpm build
   ```

## Usage

Once installed and enabled in Vencord settings, the overlay will automatically appear when you receive a DM from a friend.
